Prompt_0 – Kötelező Érvényű Fejlesztési Keretrendszer (Epubly.hu)

Feladatod egy teljesen működő webalkalmazás rekonstruálása az alábbi szigorú feltételek szerint.

Technikai megkötések (Kötelező):
1. A kód 100%-ban GitHub Pages kompatibilis legyen.
2. Kizárólag HTML, CSS és JavaScript használható.
3. TILOS TypeScript (.ts), TSX, keretrendszerek (React, Vue) vagy build eszközök (Vite, Webpack) használata.
4. Minden fájl közvetlenül futtatható legyen statikus környezetben (böngészőben).
5. Külső könyvtárak csak CDN-ről: JSZip (epub kibontás), PDF.js (pdf renderelés), QRCode.js (támogatás).

Kódszervezés:
- `index.html`: A teljes DOM struktúra, modális ablakok, sablonok.
- `style.css`: Minden stílus, CSS változók (themes), reszponzivitás.
- `app.js`: A teljes üzleti logika, Vanilla JS modul (`type="module"`), Epubly névtérben.
- `version.js`: Verziókövetés.
- `manifest.json`: PWA konfiguráció.

Ez az első lépés. Kérlek, erősítsd meg, hogy érted a keretrendszert, és várd a következő utasítást a logikai felépítésről! Ne generálj még kódot.
